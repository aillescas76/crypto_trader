# Step 5a Analysis: DualMomentumSharpeRank3w (Absolute Momentum Filter)

## Mechanism
Add absolute momentum filter (4w return > 0%) to SharpeRank3w: force cash when all gated assets have negative 4w returns simultaneously.

## Empirical Results (training period 2022-01-01 to 2024-12-31)

### Q1: Signal frequency
- 4/53 rebalance checkpoints (7.5%) where all gated assets have negative 4w returns
- SMA gate already blocks ALL assets at 33/53 checkpoints
- Filter only applies to 20 "invested" checkpoints; blocks 4 of 20 (20%)

### Q2: PnL impact
- Average 3w forward return during filter periods: **+4.41% (POSITIVE)**
- Jan 2024: filter blocked BTC (+24.63%) and ETH (+23.37%) before major rally
- Score: 4 missed recoveries vs 3 avoided losses
- Training equity: base +100.9% vs filtered +88.0% (-12.9pp loss)

### Q3: Validation (2025+)
- 3 full-cash events / 21 checkpoints: more misses than avoids
- Validation: base -25.0% vs filtered -22.8% (marginal +2.2pp)

### Q4: Threshold sensitivity
- Sharp jump at 0% (4 events vs 0 events at -5%): zero threshold is on a noise boundary
- Returns between -5% and 0% have no predictive power for next 3w direction

### Q5: Regime behavior
- 2022 bear: SMA gate blocks everything, filter is 100% redundant (0 incremental protection)
- 2023 recovery: no positions in either strategy, filter has no effect
- 2024 bull: 4 false positives, the Jan 2024 event alone dominates

### Q6: Max drawdown
- Training worst 3m: -13.3% (base) vs -13.6% (filtered) — slightly WORSE
- Validation worst 3m: -55.7% (base) vs -54.1% (filtered) — marginally better

## GATE RESULT: FAIL (Gate 2)
Average forward return during filter periods is +4.41% (must be negative to proceed)

## VERDICT: ABANDON
Structurally identical failure to Daniel-Moskowitz: redundant in deep bears (SMA gate handles it), harmful in V-shape recoveries. The threshold sits on a noise boundary — near-zero returns are coin flips for the next 3 weeks.
